# Productivity & Trading Journal App

## Project Overview
The **Productivity & Trading Journal App** is a web-based application designed to help users enhance their productivity through goal tracking, to-do lists, habit tracking, and journaling. It also includes a dedicated trading journal section for users to log their trades, including performance metrics and insights. The app features dark mode functionality and a user-friendly interface powered by Tailwind CSS and Chart.js for data visualization.

## Installation
To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/productivity-trading-journal.git
   cd productivity-trading-journal
   ```

2. **Open the `index.html` file** in a web browser. There are no additional dependencies required to run the application since it's a front-end project.

## Usage
Once the app is open in your browser, you can:

- **Login or Sign Up** using the modal that appears on the homepage.
- Navigate through the **Dashboard**, **Trackers**, **Notes**, **Trading Journal**, and **Extras** sections using the side menu or bottom navigation.
- Add daily goals, to-do items, track habits, log trades, and take notes.
- Use the Pomodoro timer for productivity sessions.

## Features
- **Daily Productivity Tracker**: Set and track daily goals and to-dos.
- **Habit Tracking**: Monitor and maintain healthy habits.
- **Mood Tracker**: Select and log your current mood.
- **Pomodoro Timer**: Enhance focus and work sessions.
- **Trading Journal**: Log trading activities with detailed metrics.
- **Notes Section**: Write and manage notes with rich text formatting.
- **Data Visualization**: View your progress with charts and statistics.
- **Dark Mode**: Toggle between light and dark themes for better visibility.

## Dependencies
This project utilizes the following libraries and frameworks:
- **Tailwind CSS**: For styling and responsive design.
- **Chart.js**: For data visualization in graphs and analytics.

Note: The libraries are linked via CDN in the `index.html` file, and no additional installation is needed.

## Project Structure
The project files are organized as follows:
```
/Productivity & Trading Journal App

### Key Components
- **index.html**: Contains the HTML structure for the app, including modals, layout, and navigation.
- **app.js**: JavaScript for managing the behavioral logic, such as loading data, managing user interactions, and functionalities across different sections.
- **styles.css**: Contains custom styles to enhance the appearance of the application.

## Conclusion
The **Productivity & Trading Journal App** is a practical tool for busy individuals looking to enhance their productivity and effectively manage their trading activities. Feel free to explore the code, modify it as per your needs, and contribute to its development.
